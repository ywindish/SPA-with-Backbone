変更点は以下の通りです。

* ライブラリを 2013/09/07 時点での最新版に差し替えた
 * jQuery 1.10.2
 * Backbone.js 1.0.0
 * Underscore.js 1.5.1
 * Handlebar.js 1.0.0
 * Bootstrap 2.3.2
* Grunt.js は使わないようにした
 * Handlbar.js テンプレートは HTML に埋め込んだ
* Twitter 検索は動作しないため、コメントアウトした

2013/09/07 最新版 Chrome にて動作確認。
